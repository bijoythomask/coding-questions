# Coding Interview Questions Project

This project is a comprehensive collection of coding interview questions, organized by topic and structured for easy navigation and study. Each topic contains multiple questions, each in its own subfolder with a Java solution and a README.md for documentation.

## Sections

- [Binary Tree Questions](./src/main/java/org/example/cs/binaryTree/)
- [Graph Questions](./src/main/java/org/example/cs/graph/)
- [Heap & Priority Queue Questions](./src/main/java/org/example/cs/heap/)
- [Linked List Questions](./src/main/java/org/example/cs/linkedlist/)
- [Other/Miscellaneous Questions](./src/main/java/org/example/cs/other/)
- [Sliding Window Questions](./src/main/java/org/example/cs/slidingwindow/)
- [Stack Questions](./src/main/java/org/example/cs/stack/)
- [Two Pointers Questions](./src/main/java/org/example/cs/twopointers/)
- [Data Structures (DS) - Queue, Sorting, Tree](./src/main/java/org/example/ds/)
- [SQL Interview Questions](./src/main/resources/)

## Structure

- Each topic folder contains a README.md listing all questions in that topic.
- Each question is in its own subfolder, with:
  - The Java solution file
  - A README.md for problem description, approach, and sample input/output
- Utility classes (e.g., ListNode, TreeNode) are in the topic root for shared access.

## How to Use

- Browse by topic using the links above.
- For each question, review the README.md and Java solution.
- Add your own notes, solutions, or test cases as you practice.
