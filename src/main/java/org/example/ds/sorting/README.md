### BubbleSort

**What is Bubble Sort?**
Bubble Sort is the simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in wrong order. It is not suitable for large datasets as its average and worst case complexity are O(n^2) where n is the number of items. Bubble sort is stable and adaptive.

**Sample Input:**
`[5, 2, 6, 1, 3, 4]`

**Sample Output:**
`[1, 2, 3, 4, 5, 6]`

**Approach:**
- Compare each pair of adjacent elements
- Swap if out of order
- Repeat until sorted
