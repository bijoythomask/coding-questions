# HighFive

## Problem Description
Given a list of scores of different students, return the average score of each student's top five scores in the order of each student's id.

Each entry items[i] has items[i][0] the student's id, and items[i][1] the student's score. The average score is calculated using integer division.

Implement the HighFive class:
- HighFive() Initializes the object with an empty list of scores.
- void add(int id, int score) Updates the list with the score of the student with id = id.
- double[][] getTopFive() Returns a list of lists of size 2, where each list contains the id and the average score of the student with the top five average scores from the high five. The average score should be integer division of the sum of the top five scores and that student's number of top five scores.
