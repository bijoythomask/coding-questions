# Heap Coding Interview Questions

This folder contains coding interview questions related to heaps and priority queues. Each question is organized in its own subfolder with the solution and documentation.

## Questions

- [FindMedianiInDatastream](./FindMedianiInDatastream)
  - **Description:** Find the median of a data stream. Implement a MedianFinder class to add numbers and return the median. Example: For arr = [2,3,4], median is 3. For arr = [2,3], median is 2.5.
- [HighFive](./HighFive)
- [KPairWithSmallestSum](./KPairWithSmallestSum)
- [KWeakestRowInMatrix](./KWeakestRowInMatrix)
- [KthLagrgestInStream](./KthLagrgestInStream)
- [LastStoneWeight](./LastStoneWeight)
- [MaximumProductOfTwoElelments](./MaximumProductOfTwoElelments)
  - **Description:** Given an array of integers, choose two different indices i and j. Return the maximum value of (nums[i]-1)*(nums[j]-1). Example: nums = [3,4,5,2] → Output: 12.
- [MeetingRoomsII](./MeetingRoomsII)
- [MergeLinkedList](./MergeLinkedList)
- [RelativeRanks](./RelativeRanks)
  - **Description:** Given an array of scores, assign ranks: "Gold Medal", "Silver Medal", "Bronze Medal", or placement number. Example: score = [5,4,3,2,1] → Output: ["Gold Medal", "Silver Medal", "Bronze Medal", "4", "5"].
- [SlidingWindowMaximum](./SlidingWindowMaximum)
- [SubSequanceOfArrayWithLargestSum](./SubSequanceOfArrayWithLargestSum)
  - **Description:** Given an integer array nums and integer k, find a subsequence of length k with the largest sum. Example: nums = [2,1,3,3], k = 2 → Output: [3,3].
