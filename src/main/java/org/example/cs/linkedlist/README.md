# Linked List Coding Interview Questions

This folder contains coding interview questions related to linked lists. Each question is organized in its own subfolder with the solution and documentation.

## Questions
- [IntersectionTwoLinkedLists](./IntersectionTwoLinkedLists)
  - **Description:** Given the heads of two singly linked lists, return the node at which the two lists intersect. If no intersection, return null.
- [LinkedListCycle](./LinkedListCycle)
  - **Description:** Return true if there is a cycle in the linked list (a node can be reached again by following next pointers), otherwise return false.
- [PalindromeLinkedList](./PalindromeLinkedList)
  - **Description:** Given the head of a singly linked list, return true if it is a palindrome, otherwise return false.

Each question folder contains:
- The Java solution file
- A README.md with the problem description, approach, and sample input/output
