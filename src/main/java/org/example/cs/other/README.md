# Other Coding Interview Questions

This folder contains miscellaneous coding interview questions. Each question is organized in its own subfolder with the solution and documentation.

## Questions
- [ChampagneTower](./ChampagneTower)
  - **Description:** Stack glasses in a pyramid, each glass holds one cup of champagne. Simulate pouring and return the amount in a specific glass.
- [DivideTwoIntegers](./DivideTwoIntegers)
  - **Description:** Divide two integers without using multiplication, division, or mod operator. Truncate toward zero and handle 32-bit signed integer overflow.
- [LongestValidParentheses](./LongestValidParentheses)
  - **Description:** Given a string of '(' and ')', return the length of the longest valid (well-formed) parentheses substring.
- [NextPermutation](./NextPermutation)
  - **Description:** Given an array of integers, find the next lexicographically greater permutation. If not possible, rearrange as the lowest possible order.
- [ReverseInteger](./ReverseInteger)
  - **Description:** Given a signed 32-bit integer, return its digits reversed. If reversing causes overflow, return 0.
- [RotateString](./RotateString)
  - **Description:** Given two strings, return true if one can become the other after some number of left shifts.

Each question folder contains:
- The Java solution file
- A README.md with the problem description, approach, and sample input/output
