# Graph Coding Interview Questions

This folder contains coding interview questions related to graphs. Each question is organized in its own subfolder with the solution and documentation.

## Questions
- [AllPathsFromSourceToTarget](./AllPathsFromSourceToTarget)
  - **Description:** Given a directed acyclic graph (DAG) of n nodes labeled from 0 to n-1, find all possible paths from node 0 to node n-1. The graph is given as graph[i], a list of all nodes you can visit from node i.
- [FindTheCelebrity](./FindTheCelebrity)
  - **Description:** At a party with n people, a celebrity is someone known by everyone but knows no one. Given a helper function knows(a, b), find the celebrity or return -1 if none exists. Example: graph = [[1,1,0],[0,1,0],[1,1,1]] → Output: 1.
- [BreadthFirstSearch](./BreadthFirstSearch)
- [DepthFirstSearch](./DepthFirstSearch)
- [NetworkDelayTime](./NetworkDelayTime)
- [NumberofProvinces](./NumberofProvinces)

Each question folder contains:
- The Java solution file
- A README.md with the problem description, approach, and sample input/output
